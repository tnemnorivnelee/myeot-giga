# PRD — 몇기가 (myeot-giga)

- **Status**: Draft
- **Last Updated**: 2026-06-30

---

## 1. 배경 및 목적

알뜰폰 eSIM 회선을 사용하는 부모님이 데이터 잔량을 확인할 방법이 없다.
해당 알뜰폰 사업자는 공식 앱이나 위젯을 제공하지 않으며, 기존 시장 앱들은 광고 과다 + 구식 UI로 부모님이 사용하기 어렵다.

광고 없이, 오직 데이터 알림이라는 본질에만 집중한 클린 앱을 직접 개발한다.

---

## 2. 타겟 플랫폼

- **Android 전용**
- iOS 제외: 시스템 API 정책상 서드파티 앱에서 기기 전체 셀룰러 데이터 조회 불가

---

## 3. 핵심 기능 명세

### 3.1 설정

| 기능 | 설명 |
|---|---|
| 요금제 리셋 날짜 | 매월 데이터가 초기화되는 날짜 (1~31일) 선택 |
| 기본 용량 설정 | 요금제 총 데이터 용량 (GB) 숫자 입력 |
| 다중 알림 임계값 | 알림 받을 수치 목록 복수 추가/삭제 가능 (예: 10GB, 15GB, 18GB) |

### 3.2 데이터 측정 및 백그라운드 감시

| 기능 | 설명 |
|---|---|
| 셀룰러 전용 측정 | Wi-Fi 트래픽 제외, 순수 모바일 데이터만 측정 |
| 측정 범위 | 당월 리셋 날짜 00:00:00 ~ 현재 시간 누적 사용량 |
| 백그라운드 감시 | 앱 종료 상태에서도 약 15분 주기로 데이터 체크 |
| 다중 알림 | 임계값 도달 시 시스템 알림 발송 (1회성, 잔류형 UX) |

알림 문구 예시: `"현재 데이터 사용량이 15GB에 도달했습니다."`

### 3.3 홈 화면 위젯

2가지 레이아웃 중 사용자 선택:

- **Option A**: 시각적 게이지(ProgressBar) + 숫자 (예: `12.5GB / 20GB`)
- **Option B**: 시각적 게이지(ProgressBar) + 퍼센트 (예: `62.5%`)

백그라운드 갱신 시 위젯도 함께 업데이트.

### 3.4 메인 대시보드

| 항목 | 설명 |
|---|---|
| 당월 누적 사용량 | 시각적 게이지 UI로 표시 |
| 오늘 사용량 | 오늘 하루 소모한 실시간 데이터량 |
| Top 3 앱 | 당월 데이터 소모량 상위 3개 앱 (앱 이름, 아이콘, 소모량) |

---

## 4. 제외 기능

| 기능 | 제외 이유 |
|---|---|
| 자동 데이터 차단 | Android 보안 정책상 서드파티 앱의 셀룰러 강제 차단 불가 |
| iOS 버전 | 시스템 API 정책상 기기 전체 데이터 조회 불가 |
| 광고 / 트래킹 | 프로젝트 핵심 원칙 위배 |

---

## 5. 비기능 요구사항

- 광고 없음
- 불필요한 백그라운드 트래킹 없음
- 부모님이 직접 사용할 수 있는 직관적인 UI
- 가볍고 빠른 앱 (최소한의 리소스 사용)
